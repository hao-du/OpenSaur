## Why

The project requires a modern frontend framework to build user interfaces. Installing React with Vite and TypeScript provides a high-performance, type-safe development environment.

## What Changes

Initialize a new React project using Vite and TypeScript in the `src/OpenSaur.CashPilot.Web/client` directory. This will add the necessary configuration files, dependencies, and a basic project structure.

## Capabilities

### New Capabilities
- `web-client-scaffold`: Initial scaffolding for the web client using React, Vite, and TypeScript.

### Modified Capabilities
- None

## Impact

Adds a new `client` directory within the `OpenSaur.CashPilot.Web` project. It introduces Node.js dependencies and configuration files (`vite.config.ts`, `tsconfig.json`, `package.json`) to the workspace.
