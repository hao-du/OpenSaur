# Feature Specification: Currencies

This specification covers the management of supported currencies.

## Business Goal
To provide a list of currencies used in transactions and exchange rate calculations.

## Requirements

### Currency Management
- **CRUD Operations**: Users can manage the list of available currencies.
- **Usage**: Currencies are fundamental to all transaction types (CashFlow, BankAccount, Transfer, Exchange).

## Acceptance Criteria
- [ ] A list of default currencies is available.
- [ ] Users can add/update currencies for their financial tracking.
