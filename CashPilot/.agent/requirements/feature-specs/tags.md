# Feature Specification: Tags

This specification covers the tagging system used for organizing and finding transactions.

## Business Goal
To provide a mechanism for users to label transactions for categorization and visual identification.

## Requirements

### Tagging System
- **Tag Management**: Users can create, update, and delete tags.
- **Transaction Tagging**: Users can apply tags to transactions to categorize them (e.g., "Food", "Rent").
- **Marker Tags**: A special type of tag used to "mark" or highlight transactions in specific views (like a calendar).

## Acceptance Criteria
- [ ] Users can create custom tags.
- [ ] Users can associate tags with transactions.
- [ ] A "Marker" tag can be used to highlight items on a calendar view.
